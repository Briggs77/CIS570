This is a Linux based Project which was developed and ran through WSL and on native Linux

To run this Project, first run ./build_ogre_linux_c++latest.sh to build the Ogre Next 3.0 Library. Then run ./build_linux.sh to build the main project.

Make sure the shell scripts have permissions with:

              chmod +x build_ogre_linux_c++latest.sh


              chmod +x build_linux.sh

              chmod +x setup_dependencies.
